# Exercício - 6
### Crie um arquivo em xml
#### Nesse exercício foi solicitado:
- Três filmes da sua preferência contendo:
  - Título
  - Ano de lançamento
  - Elenco principal
  - Gênero
  - Sinopse 
- Três livros da sua preferência contendo:
  - Título
  - Autor
  - Ano de lançamento
  - Gênero
  - Número de páginas
- Três comidas favoritas contendo: 
  - Principais ingredientes

#### Para isso, foi necessário os seguintes passos:
##### Criar um novo reposítorio
```
mkdir repositorio
cd repositorio
git init
```
##### Criar o arquivo atendendo todos os requisitos
```
touch arquivo.xml
```

```
git remote -v
git remote add 'link do respectiva projeto no seu github'
git push origin --all
```
